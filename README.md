# sn-goapi

Package for [stacker.news](https://stacker.news) API access

Install via `go get github.com/ekzyis/sn-goapi`

Supports:

- [ ] Post of type ...
  - [ ] ... Discussion
  - [x] ... Link
  - [ ] ... Bounty
  - [ ] ... Poll
- [x] Reply (comments)
- [ ] Tips
- [x] Checking dupes
- [x] Checking notifications

`SN_AUTH_COOKIE` must be set with a valid session cookie.
